# 🔐 Basic Backend Auth System (Internship Submission)

## ✨ Objective
User registration and login system using Express.js and MongoDB with JWT authentication.

## 📦 Tech Stack
- Node.js + Express.js
- MongoDB + Mongoose
- bcrypt.js for password hashing
- JWT for token-based authentication
